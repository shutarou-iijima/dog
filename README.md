## 概要
[JitPack](https://jitpack.io/)のサンプル。

build.gradleに下記を追加すると使えるようになる。
```
repositories {
    maven { url 'https://jitpack.io' }
}

dependencies {
    implementation "com.github.shutarou-iijima:dog:0.0.1"
}
```

## リポジトリに公開する方法
- 0.0.1 のような形式でタグ付 (githubの画面から)
- ビルド (JitPackの画面から)
